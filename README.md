# Настя & Тима — Save the Date

A hand-built wedding "save the date", rebuilt from a bloated Webador export into
clean **semantic HTML5 + one CSS file + one vanilla-JS file** — no framework,
no runtime dependencies.

## Run it

Static site — any of these work:

```bash
open index.html          # 1. zero setup, open in a browser
npm start                # 2. local server → http://localhost:5173 (python3, built into macOS)
```

`open index.html` works offline; serving over http (option 2) is nicer so the
Yandex map embed and Google Fonts load without `file://` quirks.

## Production build

```bash
npm run build            # minifies HTML + CSS → ./dist  (zero-dependency, node build.mjs)
npm run serve:dist       # preview the build → http://localhost:5174
```

`build.mjs` writes a minified copy of the site to `dist/`:

- **CSS** minified with a string/comment-aware pass (safe for `calc()` operator
  spaces and the inline SVG `data:` URIs).
- **HTML** comments stripped + whitespace collapsed (raw-text tags protected).
- JS + images copied as-is.
- **WebP:** if [`cwebp`](https://developers.google.com/speed/webp) is on `PATH`
  (`brew install webp`), the build also emits `.webp` copies of every raster
  image — add `<picture>` sources to serve them. (Not generated by default
  because no WebP encoder ships with macOS.)

`dist/` is a generated artifact — safe to delete and rebuild.

## Performance & A11y

- **Images:** above-the-fold hero photos `preload`ed with `fetchpriority="high"`;
  everything below the fold is `loading="lazy"` + `decoding="async"`. Photographic
  PNGs converted to JPEG (`couple` 369 KB → 109 KB). `color-palette.png` stays PNG
  (it uses transparency — convert to WebP via the build for a ~70% saving).
- **Responsive:** mobile-first; `overflow-x: hidden` guards against horizontal
  scroll; full-bleed feature images on phones.
- **A11y:** skip link, one `<h1>` + ordered h2/h3 hierarchy, `aria-labelledby`
  sections, labelled form controls, `aria-hidden` on decorative SVGs, visible
  `:focus-visible` rings, and `prefers-reduced-motion` support.

## Design system

- **Fonts:** Philosopher (headings/body, Cyrillic) + Caveat (handwritten accents)
- **Accent** `#a5655f` · **Headings** `#5d2f28` · **Text** `#624039` · **Blush bg** `#f9f2f1`

## Notes

- **RSVP backend** — the form posts to a Google Apps Script web app that emails
  the submission and appends it to a Google Sheet. Configure the endpoint via
  `RSVP_ENDPOINT` in `scripts/main.js` (empty string = demo mode, submits
  nowhere). Setup steps in [`backend/SETUP.md`](backend/SETUP.md). A honeypot
  field (`company`) blocks basic spam bots.
- **`og:image` is a relative path.** Social scrapers need absolute URLs — set the
  production domain (e.g. `https://your-domain/assets/images/couple.jpg`) before sharing.
- The page is `noindex,nofollow` (private invitation) — remove that meta tag if
  you want it indexed.
